# 🎓 Student Management System (Python Tkinter)

## 📌 Description
A GUI-based desktop application to manage student records using Python and Tkinter.

## 🚀 Features
- Add students
- View students
- Update student details
- Delete students
- Search functionality

## 🛠 Technologies Used
- Python
- Tkinter
- File Handling (.txt)

## ▶️ How to Run
1. Install Python
2. Run:
   python main.py
